# プロジェクト概要

## 目的

本プロジェクトは、モノポリーゲームを研究プラットフォームとして構築し、以下の研究を可能にする:

- 行動経済学的な意思決定パターンの分析
- ゲーム理論に基づく戦略的行動の検証
- 強化学習アルゴリズムの開発と検証

## 主要機能

### 1. ゲームプレイ機能

- 2-8人のリアルタイムマルチプレイヤー対戦
- モノポリーの公式ルール完全実装
- WebSocketによる同期通信

### 2. データ収集機能

- 全プレイヤーアクションの自動記録
- イベント単位での詳細な行動履歴保存
- 状態遷移の完全なトレーサビリティ

### 3. 分析機能

- 行動経済学分析（リスク選好、損失回避、サンクコスト効果等）
- ゲーム理論分析（ナッシュ均衡、協力vs競争、支配戦略等）
- 時系列分析（資産推移、勢力変化、マーケットシェア等）
- 強化学習用データエクスポート（OpenAI Gym、TensorFlow形式）

### 4. AI対戦機能

- 強化学習エージェントとの対戦

## 技術的目標

- **データ完全性**: 全てのゲームイベントを欠落なく記録
- **保守性**: 長期的な拡張・改修に耐える設計
- **再現性**: 任意の時点のゲーム状態を再現可能
- **分析容易性**: 研究目的のクエリを効率的に実行可能
